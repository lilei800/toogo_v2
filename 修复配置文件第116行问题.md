# 修复配置文件第116行问题

## 🔍 问题根源

**第116行的description字段缺少结束引号！**

```yaml
description: "Toogo.Ai 鍏ㄨ嚜鍔ㄨ櫄鎷熻揣甯侀噺鍖栦氦鏄撳钩鍙?
```

这行没有结束引号 `"`，导致YAML解析错误。

## ✅ 解决方案

### 在服务器上执行：

```bash
# 停止服务
sudo systemctl stop toogo

# 修复第116行（添加结束引号）
cd /opt/toogo/toogo_v2/server/manifest/config
sed -i '116s/$/"/' config.yaml

# 或者直接替换整行
sed -i '116s/.*/  description: "Toogo.Ai 全自动虚拟货币量化交易平台"/' config.yaml

# 验证修复
sed -n '115,118p' config.yaml

# 启动服务
sudo systemctl start toogo
sudo journalctl -u toogo -f
```

### 或者手动编辑：

```bash
vim /opt/toogo/toogo_v2/server/manifest/config/config.yaml
```

找到第116行，修改为：
```yaml
  description: "Toogo.Ai 全自动虚拟货币量化交易平台"
```

保存并退出（`:wq`）
