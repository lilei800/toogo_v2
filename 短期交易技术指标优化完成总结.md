
**修改后**：
```go

    score += 0.5
}

    score += 0.3

// 判定阈值：0.2（降低阈值，更敏感）

    return "up", score
```

- ✅ 使用EMA8-EMA12判断趋势（更敏感）
- ✅ 价格与EMA8比较（更快反应）
