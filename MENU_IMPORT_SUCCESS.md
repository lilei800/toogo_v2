## 🐛 故障排查

### Q1: 看不到新菜单

**解决方案**:
```
1. 强制刷新浏览器 (Ctrl + F5)
2. 清除浏览器缓存
3. 退出并重新登录
4. 检查角色权限配置
```

### Q2: 点击菜单报404

**原因**: 前端路由未生效

**解决方案**:
```
1. 确认前端服务正常运行
2. 检查路由文件是否存在:
   - web/src/router/modules/trading.ts
   - web/src/router/modules/payment.ts
3. 刷新前端服务
```

### Q3: 菜单显示但没权限

**解决方案**:
```sql
-- 检查菜单权限
SELECT id, title, permissions FROM hg_admin_menu 
WHERE title IN ('量化交易', 'USDT管理');

-- 配置权限后重新登录
```

### Q4: 图标不显示

**解决方案**:
```sql
-- 更新图标
UPDATE hg_admin_menu SET icon = 'WalletOutlined' WHERE name = 'trading';
UPDATE hg_admin_menu SET icon = 'DollarOutlined' WHERE name = 'payment';
```
