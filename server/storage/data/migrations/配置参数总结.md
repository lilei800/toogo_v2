## 🔍 配置特点说明

### 1. 阈值设置（BTCUSDT 和 ETHUSDT 相同）

- **LowV = 0.5**：实际波动小于正常波动的一半 → 低波动
- **TrendV = 1.2**：实际波动大于正常波动的1.2倍且方向一致 → 趋势
- **HighV = 2.0**：实际波动大于正常波动的2倍但方向不一致 → 高波动
- **DThreshold = 0.7**：趋势需要较强的方向一致性（70%以上）

### 2. Delta值差异

- **BTCUSDT**：Delta值较大（30-600 USDT），因为价格高，波动绝对值大
- **ETHUSDT**：Delta值较小（20-400 USDT），约为BTCUSDT的2/3

### 3. 权重设置（BTCUSDT 和 ETHUSDT 相同）

- **15分钟周期权重最大（30%）**：核心周期，最能反映中期趋势
- **1分钟和1小时权重较小**：避免短期噪音和长期滞后影响
- **5分钟和30分钟权重中等**：平衡短期和中期信息

## 📝 使用建议

1. **初始使用**：直接使用推荐配置
2. **观察效果**：运行1-2周，观察市场状态判断的准确性
3. **微调参数**：
   - 如果经常误判为高波动 → 适当增大Delta值
   - 如果经常误判为低波动 → 适当减小Delta值
   - 如果趋势识别不敏感 → 适当降低TrendV或DThreshold
   - 如果趋势识别过于敏感 → 适当提高TrendV或DThreshold
4. **记录优化**：记录每次调整的效果，持续优化
